# BizRate（ビジレート）

AIがビジネスアイデアを構造的に評価し、「売れる可能性」をスコア化するWebサービス。

---

## ■ 概要

ユーザーがビジネスアイデアを入力すると、
市場性・競争・収益性・実行可能性などの観点からスコアリングし、
改善方向まで提示します。

---

## ■ 背景

多くのアイデアは「なんとなく良さそう」で判断されがちですが、
本サービスはそれを構造的に分解し、意思決定を支援することを目的としています。

---

## ■ 主な機能

* ビジネスアイデアのスコア評価（100点満点）
* 5軸分析（市場性 / 競争 / 収益性 / 実行難易度 / AI代替耐性）
* 改善方向の提示
* 収益シミュレーション
* PDF出力（レポート化）

---

## ■ 技術スタック

* Next.js
* TypeScript
* Vercel（デプロイ）
* OpenAI API（分析ロジック）
